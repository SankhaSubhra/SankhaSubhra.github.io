Current and Past Research Problems

Class Imbalanced Classification
Improve immunity of Support Vector Machines, k-Nearest Neighbor, and Deep Neural Network classifiers against the detrimental effects of class imbalance.
Theoretical Analysis of Classifiers
Mathematically investigate how class imbalance affects the performance of a classifier as well as the popular evaluation indices.
Differential Evolution (DE)
Design variants of DE for large scale and noisy real valued global optimization problems. Apply DE to address problems like parameter optimization in classifiers, adversarial attacks on Deep Neural Network classifiers, etc.
Classification in Presence of Label-based Ambiguity
Formally define as an umbrella problem for various class label-based ambiguities such as noisy labels, crowd-sourced labels, unreliable experts, etc. Design classifiers which can effectively resolve such ambiguities.
